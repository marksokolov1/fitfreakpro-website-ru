# Русский стиль FitFreak Pro

## Голос бренда

FitFreak Pro говорит как практичный продукт для тренеров: прямо, спокойно и без громких обещаний. Тон — профессиональный, но не канцелярский. Главная задача текста — быстро объяснить тренеру, как приложение помогает организовать работу с клиентами.

## Предпочтительная терминология

- `тренер`, `персональный тренер`, `онлайн-тренер`
- `клиент`, `приглашенный клиент`
- `библиотека упражнений`
- `свои видео упражнений`
- `назначать тренировки`, `назначенные тренировки`
- `цели по питанию`
- `данные прогресса`, если речь идет о данных, которые клиент отправляет тренеру
- `панель тренера`
- `доступ клиента` или `клиентский доступ`

## Чего избегать

- Не использовать «тренировочный процесс» как универсальную замену конкретным действиям.
- Не писать «путь тренера», «аккуратный процесс», «система тренера» без конкретики.
- Не писать «информация о прогрессе», если лучше сказать «данные прогресса».
- Не использовать «в одном приложении» в каждом блоке.
- Не добавлять обещания экономии времени, роста дохода или результата клиента без подтвержденных данных.
- Не использовать английские фразы, кроме названий платформ, продукта, URL и технических терминов.

## «Е» и «ё»

По умолчанию используем `е`, чтобы текст выглядел современно и единообразно. `ё` допустима только там, где без нее может возникнуть неоднозначность.

## Заголовки и регистр

- Заголовки пишутся обычным предложным регистром: `Как тренер начинает работу`, а не `Как Тренер Начинает Работу`.
- `FitFreak Pro`, `App Store`, `Google Play`, `iPhone`, `Android`, `LinkedIn` сохраняются в оригинальном написании.
- Валюта: `1490 ₽ в месяц`, `0 ₽`.

## CTA

CTA должен говорить о понятном действии:

- `Скачать приложение`
- `Запросить демонстрацию`
- `Посмотреть цены`
- `Связаться с поддержкой`

Не использовать CTA вроде `Начать путь`, `Построить систему`, `Разблокировать рост`.

## Тренер и клиент

Тренер — основной покупательский пользователь сайта. Клиент использует приложение после приглашения тренера и оплачивает клиентский доступ. Текст не должен звучать как универсальное фитнес-приложение для самостоятельных тренировок.

## Цена

- Аккаунт тренера: `0 ₽`.
- Клиентский доступ: `1490 ₽ в месяц`.
- Не описывать возвраты, налоги, продление или отмену подписки без подтвержденных правил.
