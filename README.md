# Сайт FitFreak Pro на русском языке

Это статическая русская версия сайта FitFreak Pro для аудитории в Российской Федерации. Проект остается обычным HTML, CSS и vanilla JavaScript без сборки.

## Страницы

- `/` Главная страница
- `/how-it-works/` Как работает процесс тренера
- `/for-personal-trainers/` Страница для персональных тренеров
- `/pricing/` Цены
- `/about/` О продукте и основателе
- `/privacy/` Политика конфиденциальности
- `/support/` Поддержка
- `/terms/` Обзор условий, требующий юридической проверки
- `/robots.txt` Инструкции для поисковых роботов
- `/sitemap.xml` Карта публичных страниц

## Размещение

Загрузите содержимое этой папки в корень русской версии сайта, например на `/ru/` домена `fitfreakpro.com` или на отдельный домен для русской локализации.

Сайт использует корневые пути вида `/assets/css/styles.css`. Если русская версия будет размещена не в корне домена, проверьте конфигурацию хостинга и маршрутизацию ассетов перед публикацией.

В репозитории нет пакетного менеджера, фреймворка, сборки или тест-раннера. Это статический сайт на HTML, CSS и JavaScript.
